# Prancheta Virtual Smr Nrt (ANDROID)
###### Aplicativo desenvolvido para auxiliar no gerenciamento de serviços executados pelos eletricista em campo.

## Tecnologias utilizadas:
- ANDROID STUDIO
- SQLiteDatabase
- SQLiteOpenHelper
- Serializable
- BroadcastReceiver
- BaseAdapter
- ImageView
- TextView 
- ViewGroup
- LayoutInflater
- SimpleDateFormat
- ArrayList
- List
